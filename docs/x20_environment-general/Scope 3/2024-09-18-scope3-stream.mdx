기업의 가치사슬을 파악하고 관리하기 위해서는 Scope 3의 파악이 중요합니다. <br/>
기업의 전체 배출에서 Scope 3에서의 배출이 70~90% 이상을 차지한다는 사실을 감안할 때, 스코프3 관리는 배출량 감축 목표를 달성하기 위해서는 어떻게든 관리를 해야하는 범위입니다

​

이번 시리즈에서는 가치사슬 내 온실가스 관리와 관련된 Scope 3의 개념, 용어 정의, 산정 방법을 설명합니다.
관리자는 지속가능성 보고서를 이해하고, 감축 전략과 위험 관리에 이를 적용하는 데 필요한 지식을 얻게 될 것입니다. 실무자는 Scope 3 배출을 관리하는 구체적인 방법을 이해할 수 있을 것입니다.

운영 경계에 따라 Scope 1, 2와 함께 Scope 3을 구분합니다. <br/>
스코프1은 직접배출로 보고기업의 통제 안에서 일어나는 배출입니다. <br/>
스코프2는 간접배출로 외부에서 구입한 전력, 열에 의한 배출입니다. <br/>


## Scope 3란?
Scope 1,2를 제외한 보고기업의 통제 밖에 있는 가치사슬에서 일어나는 배출이 Scope 3입니다. <br/>
Scope 3은 총 15가지 카테고리로 구분합니다. <br/>
내 기업의 제품이나 서비스를 생산하는 과정을 업스트림 활동이라 합니다. <br/>
내 기업의 제품이나 서비스가 내 손을 떠나 판매하고, 사용하고 폐기하는 과정, 다른 기업에 투자하는 과정을 다운스트림 활동이라고 합니다. <br/>

​
## Scope 3 - 업스트림
업스트림 활동에서 어떤 배출활동을 포함하는지 차근차근 짚어보겠습니다. <br/>
기업이 원자재나 설비, 즉 자본재를 구매하여 제품이나 서비스를 생산하는 과정이 카테고리 1,2로 원자재와 설비의 cradle to gate 배출량을 알아야합니다. <br/>
시설을 운영하는 과정에서 사용하는 에너지의 채굴, 운반, 정제 과정의 온실가스 배출이 카테고리 3, 연료와 에너지 활동 배출에 해당됩니다. <br/>
우리 회사에 필요한 원자재와 설비를 우리 회사로 운송해오는 과정에서의 배출이 카테고리4, 업스트림 운송 배출이고, <br/>
우리 회사가 배출한 폐기물을 다른 소각장이나 매립장에서 처리하는 과정에서의 배출이 카테고리 5 배출, 임직원의 출장에서 발생하는 배출이 카테고리 6, 출장 배출, 통근에서 발생하는 배출이 카테고리 7, 통근 배출입니다. <br/>
마지막으로 회사가 임차한 사무실이나 공장의 사용에서 발생하는 배출이 카테고리 8, 업스트림 임차자산 배출입니다. <br/>

​
## Scope 3 - 다운스트림
이제 우리의 제품이 우리 손을 떠나, 다운스트림 활동으로 넘어가겠습니다. <br/>
생산한 제품과 서비스가 판매되어 우리회사에서 다른 회사로 운송될 때, <br/>
외부 서비스를 이용한다면 그 과정에서의 배출이 카테고리 9, 다운스트림 운송과 유통 배출입니다. <br/>
우리가 생산한 제품이 완제품이 아니어서 가공이 필요하다면 그 제품이 가공되는 과정에서의 배출이 카테고리 10, <br/>
판매된 제품 가공 배출이 되고 완제품이라면 판매한 제품이 사용하는 과정에서의 배출이 카테고리 11 배출이 됩니다. <br/>
그리고 최종적으로 판매한 제품을 수명종료처리, 폐기하는 과정에서의 배출이 카테고리 12 판매된 제품의 수명종료처리 배출이 됩니다. <br/>
회사가 다른 업체에게 임대해준 자산에서의 Scope 1, 2 배출은 카테고리 13, 다운스트림 임대 자산 배출입니다. <br/>
우리 회사의 제품과 서비스를 판매하는 과정에서 프랜차이즈의 배출이 카테고리 14, 프랜차이즈 배출입니다. <br/>
마지막으로 우리 회사가 다른 회사에 투자 했다면 해당 기업의 배출량은 카테고리 15, 투자 배출이 됩니다. <br/>
