하나루프의 EcoLoop 지속가능성 플랫폼으로 기업/시업장 전력사용량을 한전에서 자동으로 수집하여 실시간으로 세밀하게 전력사용 추세를 분석할 수 있습니다. 
실무자 입장에서는 데이터를 자동으로 수집하여 업무가 간소해집니다.

<CaptionedImage imageSrc="/images/posting/ecoloop/ecoloop_electricity_chart.png" containerClassName="shadow-xl"
caption="EcoLoop의 전력 분석" />

<br/><br/>

**EcoLoop 지속가능성 플랫폼을 한전과 연동하여 귀사의 전력사용 데이터를 자동으로 수집하려면 귀사의 정보제공 동의가 필요합니다. 동의는 1년간 유효합니다.**

<br/>

한전 전력사용량 정보제공을 동의하는 절차를 요약해보면 아래와 같습니다. <br/>

> <a href="https://en-ter.co.kr/" target="_blank">에너지 마켓플레이스(https://en-ter.co.kr/)</a>,<br/> <a href="https://pp.kepco.co.kr/" target="_blank">파워플래너(https://pp.kepco.co.kr/)</a>,<br/> <a href="https://evc.kepco.co.kr/" target="_blank">캡코플러그(https://evc.kepco.co.kr/)</a> 중 한개의 한전 사이트에 **회원가입이 먼저 되 있어야 합니다**. 

<br/>

1. [한전 에너지 마켓플레이스](https://en-ter.co.kr/)에 로그인 <br/>
  - 옵션1: **회원** 탭에서 개인 또는 법인 아이디로 로그인 <br/>   
  - 옵션2: **그 외 한전홈페이지 회원**(PowerPlanner 또는 KEPCO PLUG) 탭에서 해당 아이디로 로그인  
2. [한전 전력데이터 서비스마켓](https://edsm.kepco.co.kr/)에서 "전체서비스" 메뉴 > "서비스 검색" 창에서 "에코루프" 선택
3. "정보제공동의" 클릭

자세한 절차는 다음과 같습니다
