## The Looplex Services

At our imaginary company Looplex, we provide services that exposes Open API for rapid integration.

The following are the steps to connect to an API and consume services:

1. Browse the list of APIs in the directory
2. Obtain the OpenAPI specification
3. Test client locally with [generated server mocks](https://github.com/stoplightio/prism) 
3. Obtain an API key for the DEV environment
4. Do integration with on DEV environment
5. Obtain an API key for the PROD environment
6. Go LIVE!
