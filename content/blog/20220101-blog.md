Next Dev Portal has built-in blogging feature.

Just create an `*.mdx` file under `/content/blog` directory. 
Then run the generator too `yarn gen:collections` and you are good to go!

Regular blog authors can be added to `authors.yml`. (Thanks Docusaurus for the idea!)

<!--truncate-->

You can truncate the content with `<!--` `truncate` `-->` (all characters together) 
to break long text.

And of course, use Markdown + JSX!
