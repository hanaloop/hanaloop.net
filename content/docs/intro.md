## Introduction

The Next Dev Portal allows you to build websites fast.

It is mainly intended for creating (software) product documentation, but it can easily be adapted to serve any kind of website.

Content source files are written in markdown (MDX) and placed under content directory.


You can easily do syntax highlighting as seen below.
```js
const test = {
  "data": "test"
}
```
